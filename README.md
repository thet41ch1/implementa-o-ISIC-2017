# implementa-o-ISIC-2017
Implementação em Python de um modelo de Deep Learning para classificação de lesões cutâneas utilizando o dataset ISIC-2017. O código realiza o pré-processamento das imagens, organização em conjuntos de treino, validação e teste, treino do modelo e avaliação através de métricas como accuracy, loss, matriz de confusão e curva ROC.
